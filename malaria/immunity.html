Immunity¶

The immunity.py file contains a params dictionary defining the default immunity parameters. This file also holds utility function to create immunity overlays or initialize immunity based on habitat scaling.

dtk.malaria.immunity.add_immune_init(cb, site, x_temp_habitats, directory=None)[source]¶

Initializes the immunity based on habitat scaling.

Simply create the tags and call the add_immune_overlays function.

Parameters:
  • cb – The DTKConfigBuilder holding the configuration
  • site – If the site is specified, the files will be expected to be found in the immune_init/site subdirectory.
  • x_temp_habitats – List of temp habitats
  • directory – Main directory where the ..._immune_init_x_...json files are stored
Returns:

Nothing
dtk.malaria.immunity.add_immune_overlays(cb, tags, directory='E:\\Eradication\\Data_Files\\', site=None)[source]¶

Add an immunity overlay.

To do so, reads the demographics files and find the corresponding immunity initialization overlay.

Parameters:
  • cb – The DTKConfigBuilder holding the configuration
  • tags – List of immunity tags that have corresponding initialization files
  • directory – Main directory where the ..._immune_init_x_...json files are stored
  • site – If the site is specified, the files will be expected to be found in the immune_init/site subdirectory.
Returns:

Nothing
