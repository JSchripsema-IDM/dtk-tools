Geography¶

This file holds the different available geographies and methods to set the parameters relative to the climate files appropriately. The geographies dictionary associate a given geography with its parameters defining the different climate files and parameters along with specific demographics parameters.

Currently supported geographies include:

  • Garki_Single
  • Namawala
  • Sinazongwe
  • GwembeSinazongwePopCluster
  • Dielmo
  • Ndiop
  • Thies
  • Mocuba
  • West_Kenya
  • Solomon_Islands
  • Solomon_Islands_2Node
  • Nabang
  • Tha_Song_Yang
  • Malariatherapy
  • Birth_Cohort
dtk.generic.geography.convert_filepaths(params)[source]¶

Make sure to add the geography folder in front of the paths. For example, if the geography is Namawala, we want the climate/demographics files to be Namawalafile.json

Parameters:params – The config parameters
Returns:Nothing
dtk.generic.geography.get_converted_paths_for_geography(geography)[source]¶

Returns the converted paths for a given geography.

Simply copy the geography specific parameters and send it to the :any:’convert_filepaths’ function to get the correct paths. Then return the updated parameters.

Parameters:geography – The selected geography
Returns:parameters with the correct path
dtk.generic.geography.get_geography_parameter(geography, param)[source]¶

Return a particular parameter for a given geography. This function will return the parameter value with the updated path and can accommodate geography of the form: Sinazongwe.static. The .static will be removed.

Parameters:
  • geography – The desired geography
  • param – The parameter we want to retrieve
Returns:

The value of the parameter for the given geography
dtk.generic.geography.set_geography(cb, geography, static=False, pop_scale=1)[source]¶

Set the demographics in a given DTKConfigBuilder for the selected geography. The demographics can be set to static and created with the set_static_demographics function or it can be created with the population scale parameter passed.

Note

The population scale will act on the x_Birth configuration parameter if the Birth_Rate_dependence is set to a FIXED_BIRTH_DATE. It will also act on the Base_Population_Scale_Factor.

Parameters:
  • cb – The DTKConfigBuilder containing the current configuration
  • geography – The selected geography
  • static – If True, will create a static demographics. if False, will use the pop_scale
  • pop_scale – Used if the demographics is not static and will
Returns:

Nothing
