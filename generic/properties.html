Properties¶

This helper file is used for easily defining properties in the model (see Configure heterogeneous disease transmission)).

dtk.generic.properties.add_properties_overlay(cb, properties, directory=None, tag='')[source]¶

Creates a property overlay with the given properties.

Parameters:
  • cb – The DTKConfigBuilder containing the configuration
  • properties – The list of properties to add
  • directory – The directory where the demographics file is stored
  • tag – Properties tag
Returns:

Nothing
dtk.generic.properties.init_access(**kwargs)[source]¶

A shortcut function allowing to set the initial distribution of the Accessibility property.

init_access(Easy=0.5, Hard=0.5)
Parameters:kwargs – A list of key,value pairs representing the property value and initial distribution
Returns:A dictionary representing the property ready for DTK
dtk.generic.properties.set_property(property, values, initial)[source]¶

Set the value of a property.

set_property('Accessibility', ['Easy', 'Hard'], [0.5, 0.5])
Parameters:
  • property – The property to set
  • values – A list of different values this property can take
  • initial – The initial distribution
Returns:

A dictionary representing the property ready for DTK
