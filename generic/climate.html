Climate¶

This file holds the generic defaults for the climate part of the simulation. It contains a params dictionary setting the default weather values.

dtk.generic.climate.set_climate_constant(cb, **kwargs)[source]¶

Set the climate to constant weather by changing the Climate_Model parameter to CLIMATE_CONSTANT. Also set the extra parameters passed.

Parameters
  • cb -- The DTKConfigBuilder object containing the current configuration
  • kwargs -- Extra parameters to change in the config file
Returns

Nothing
